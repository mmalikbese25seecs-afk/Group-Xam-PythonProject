Project Description:

This project contains two programs:

2. Smart Calculator Function

3\.	Using GITHUB platform to merge both functions



1.NUMBER GUESSING GAME

The computer randomly selects a number between 1 and 50.



The user tries to guess the number, receiving feedback:



"Too High" if the guess is higher than the number.



"Too Low" if the guess is lower than the number.



The game keeps track of the number of guesses.



When the user guesses correctly, it displays the correct number and total attempts.



After each round, the user can choose to play again or exit from the program
